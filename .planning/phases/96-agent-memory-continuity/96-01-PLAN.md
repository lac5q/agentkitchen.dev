## Goal

Replace AgentMemory-style local context capture with a MemRoOS-native capture and handoff contract so coding work can move between Codex, Claude Code, Hermes/OpenClaw, OpenCode, Gemini/Qwen CLI, and similar agents without manual reconstruction.

## Verification Strategy

1. Unit tests prove coding-agent captures write sealed raw artifacts and durable candidates.
2. Handoff tests prove a second agent can receive task state, decision intent, sources, files, errors, and verification history.
3. Redaction tests prove secret-like content is removed from handoff packs.
4. Duplicate tests prove repeated captures are suppressed by stable hash.
5. Typecheck proves the API and schema compile.

## Tasks

### Task 1: Schema

Add:
- `agent_session_captures`
- `agent_memory_candidates`
- `agent_handoff_packs`

### Task 2: Capture Library

Implement `captureCodingAgentSession()` to normalize coding-agent session state, write sealed raw vault artifacts, redact handoff-visible fields, extract durable memory candidates, and suppress duplicates.

### Task 3: Handoff Library

Implement `buildCodingAgentHandoffPack()` to assemble "continue this work" context packs from task id, session id, or source agent id.

### Task 4: API

Add operator-gated routes:
- `POST /api/agent-memory/capture`
- `POST /api/agent-memory/handoff`

### Task 5: Verification

Add library and route tests covering capture, handoff, redaction, and duplicate suppression.
