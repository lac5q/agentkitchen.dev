# Phase 27 Plan: Progressive MCP Tool Attention

## Objective

Add a progressive MCP/tool-attention layer so agents can discover the right tool surface without loading every tool definition up front.

## Scope

- Import the progressive Knowledge MCP package into `services/knowledge-mcp`.
- Add a `tool-attention` workspace with catalog, discover, load, record outcome, and stats actions.
- Add a Kitchen API endpoint for tool attention.
- Add Kitchen client types and hook support.
- Surface tool attention in Cookbooks and Flow.

## Tasks

1. Import `knowledge_system` and tests into `services/knowledge-mcp`.
2. Extend `knowledge_workspace_call` for the `tool-attention` workspace actions.
3. Add a curated tool catalog and outcome log path.
4. Add `/api/tool-attention`.
5. Add `ToolAttentionPanel` to Cookbooks after Health Overview.
6. Add a Tool Gateway node and detail panel stats to Flow.
7. Validate missing catalog/log cases degrade cleanly.

## Validation

- `python -m pytest services/knowledge-mcp/tests`
- `npm --prefix apps/kitchen run test -- --run`
- `npm run build`
- Live `GET /api/tool-attention`
