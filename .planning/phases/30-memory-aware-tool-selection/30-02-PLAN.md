# Phase 30 Plan 02: Similar-Task Recommendations

## Goal

Extend discovery with safe, task-scoped context so agents can prefer tools that worked for similar work patterns.

## Scope

- Accept an optional context pack in discovery:
  - `task_type`
  - `repo`
  - `agent_id`
  - `tags`
- Aggregate context signals from recorded outcome metadata.
- Add context-aware boosting to discovery ranking.
- Emit `similarTaskRecommendations` without returning raw task text.
- Wire context through both top-level MCP tool wrappers and workspace action dispatch.

## Non-Goals

- No direct mem0 retrieval in this step.
- No UI changes in this step.
- No external model-based ranking.

## Verification

- Extend Knowledge MCP tests for context-pack ranking and recommendation output.
- Verify stats remain public-safe (no raw task text leakage).
