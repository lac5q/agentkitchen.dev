# Roadmap: Agent Kitchen v1.6 Monorepo + Progressive MCP Tool Attention

## Summary

v1.6 turns Agent Kitchen into the canonical monorepo and adds progressive MCP tool-attention infrastructure so agents can discover and load capabilities on demand.

## Phases

- [x] Phase 26: Monorepo Foundation (1/1 plans) — completed 2026-04-30
- [x] Phase 27: Progressive MCP Tool Attention (1/1 plans) — completed 2026-04-30
- [x] Phase 28: Monorepo CI and Deploy Hardening (1/1 plans) — completed 2026-04-30

## Outcomes

- Kitchen moved to `apps/kitchen`.
- Memory, Knowledge MCP, and voice runtimes live under `services/`.
- Root scripts delegate into the Kitchen app.
- Root `.mcp.json` points at the monorepo memory and knowledge services.
- Knowledge MCP includes progressive workspaces and a `tool-attention` action surface.
- Kitchen surfaces Tool Attention on Cookbooks and Flow.
- Production LaunchAgent deploy verified on port `3002`.
- Monorepo CI added for Kitchen and Python service validation.
