# Example Client Recall Anchor

This public-safe example shows how a deployment can define a critical memory that agents must never lose.

## Source-Backed Decision

Example Client decided on January 15, 2026 that the billing workflow may produce draft recommendations, but any customer-impacting billing action requires human approval before execution.

## Required Recall Terms

- Example Client
- billing workflow
- human approval
- source-backed

## Why This Exists

This anchor lets operators test whether a critical source-backed decision is retrievable from the filesystem, QMD, and, when enabled, mem0 or the app recall surface. Replace this example with private tenant-specific anchors during onboarding.
