# MemroOS launch media pack

Assets for the v0.9 launch push.

## Files

- `memroos-v0.9-demo-64s.mp4`: 64-second demo clip for X and LinkedIn.
- `social-01-memory-search.png`: social card for retained memory search.
- `social-02-dispatch-console.png`: social card for governed dispatch.
- `social-03-workflow-map.png`: social card for workflow proof trail.
- `memroos-architecture-card.png`: simple architecture graphic.
- `memroos-architecture-card.svg`: editable source for the architecture card.
- `first-workflow-teaser-14s.mp4`: short first-workflow teaser clip.
- `first-workflow-teaser-10s.gif`: GIF version of the teaser.

## Suggested use

Use the 64-second demo clip on the main X and LinkedIn launch posts. Use the three social cards as follow-up posts, carousel images, or replies. Use the architecture card when explaining that MemroOS is a context plane before dispatch and proof trail after execution.
